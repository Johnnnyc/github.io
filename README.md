# 个人网站

这是我的个人网站项目，使用 React + TypeScript + Vite 构建。

## 技术栈

- React 18
- TypeScript
- Vite
- Framer Motion (动画效果)
- Lucide React (图标)

## 项目结构

```
src/
  components/      # 组件
  styles/          # 样式
  App.tsx          # 应用主组件
  main.tsx         # 应用入口
```

## 运行项目

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建项目

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

## 部署

项目使用 GitHub Actions 自动部署到 GitHub Pages。
